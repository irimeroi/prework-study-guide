HTML

  • HTML stands for Hypertext Markup Language.
  • It's the standard markup language used to structure a webpage.
  • Contains semantic and meta-related elements that provide a strong foundation of the web content.

CSS

  • CSS stands for Cascading Style Sheets.
  • It's the mechanism used to style HTML documents.
  • The easiest and most common way to style a webpage with CSS is using an external style sheet.

Git

  • Git stands for Global Information Tracker
  • It allows developers to work on a project simultaneously without interefering with each other
  • Ex. "git status checks what branch we are currently on", or "git checkout -b branch-name" creates a new branch and switches to it.
